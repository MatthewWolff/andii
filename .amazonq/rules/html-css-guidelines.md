# HTML/CSS Guidelines

## Structure
- Single-page application
- Responsive design using Bootstrap 5.3.0
- Mobile-first approach

## Styling
- Primary background: #CCCCFF (purple)
- Secondary accent: #F3B2CB (pink)
- Custom fonts: Bubblegum Sans, Quantico
- Bootstrap utilities for layout and spacing

## Best Practices
- Keep all styles inline or in `<style>` tag
- Use semantic HTML
- Ensure accessibility with proper alt text
- Maintain responsive design for all screen sizes
- Prefer to use Bootstrap elements instead of implementing new

## Performance
- Lazy load images in slideshows
- Preload next 2 images in background
- Only render loaded images
